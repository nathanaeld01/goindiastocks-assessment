### Run Locally
Clone the project
```bash
  git clone https://github.com/nathanaeld01/goindiastocks-assessment.git
```

Go to the project directory
```bash
  cd goindiastocks-assessment
```

Install dependencies
```bash
  npm install
```

Start the server
```bash
  npm run start
```
